# 🚀 CodSoft Data Science Internship Projects

This repository contains my completed projects for the **CodSoft Data Science Internship**. These projects demonstrate the application of Machine Learning techniques using Python for classification and regression problems.

## 📌 Completed Tasks

### ✅ Task 1: Titanic Survival Prediction
- Built a machine learning model to predict whether a passenger survived the Titanic disaster.
- Performed data cleaning, preprocessing, exploratory data analysis (EDA), feature engineering, model training, and evaluation.

### ✅ Task 3: Iris Flower Classification
- Developed a classification model to identify Iris flower species based on sepal and petal measurements.
- Implemented data visualization, preprocessing, model training, and performance evaluation.

### ✅ Task 4: Sales Prediction Using Python
- Built a regression model to predict product sales based on advertising data.
- Applied data preprocessing, visualization, model training, and prediction techniques.
